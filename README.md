# Izdevumu izsekotājs 

 

Python aplikācijas, kas paredzēta personīgo izdevumu uzskaitei un analīzei. 

 

## Uzstādīšana 

 

git clone https://github.com/jelenavalkovska-cmyk/05-projekts.git

cd expense-tracker 

python expense_tracker/app.py 

 

Nav nepieciešamas papildus bibliotēkas — tikai Python 3.10+. 

 

## Lietošana 

 

Programma darbojas interaktīvā režīmā ar izvēlni: 

1) Pievienot izdevumu 

2) Parādīt izdevumus 

3) Filtrēt pēc mēneša 

4) Kopsavilkums pa kategorijām 

5) Dzēst izdevumu 

6) Eksportēt CSV 

7) Iziet
